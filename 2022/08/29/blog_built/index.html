安装依赖

  • node.js

node.js获取

记得添加环境变量再next

  • git

git获取
一路next就行
验证是否安装成功：
验证git: 桌面右键有git bash

  • 验证node.js: 点击git bash依次执行命令

$ node -v
$ npm -v

输出版本号就算成功

  • 安装cnpm

git bash中执行： 

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org

检查是否安装完成：

$ cnpm -v

安装hexo并初始化

  • hexo

 $ cnpm install hexo-cli -g

  • 创建blog文件夹并在当前目录打开git bash

依次执行命令：

$ hexo init
$ cnpm install

这时可以在本地浏览我们的博客了

$ hexo s

可以更换主题然后配置文件啦！
